# 태그를 사용하여 옵트인 구성 {#configuring-opt-in-with-launch}

태그를 사용하여 옵트인용 CX 엔터프라이즈 솔루션을 간편하게 사용할 수 있습니다.

## 태그를 사용하여 옵트인 시나리오 구성 {#section-8aa1b58bf8374c938aa8cfdeddbad6ff}

Adobe Experience Platform 데이터 수집의 [태그](https://experienceleague.adobe.com/docs/experience-platform/tags/home.html?lang=ko)을(를) 사용하면 Adobe 솔루션을 사용하여 시나리오에 옵트인을 쉽게 구성하고 설정할 수 있습니다. Analytics, Target, Audience Manager 등을 활성화하여 CX 엔터프라이즈 솔루션에 대한 방문자 옵트인 동의를 수집하거나, 동의 관리 시스템에 옵트인할 CX 엔터프라이즈 솔루션을 모두 선택할 수 있습니다.

**[!UICONTROL Experience Cloud ID Service] 태그 확장 구성**

[!UICONTROL Experience Cloud ID Service] 태그 확장이 아직 설치되지 않은 경우 속성을 연 다음, *확장* > *카탈로그*&#x200B;를 클릭하고 [!UICONTROL Experience Cloud ID Service] 태그 확장을 마우스로 가리킨 다음 *설치*&#x200B;를 클릭합니다.

확장 기능을 구성하려면 *확장* 탭을 열고 확장 위로 마우스를 가져갑니다. 그런 다음 *구성*&#x200B;을 클릭합니다.

추가 참조 정보는 [!UICONTROL Experience Cloud ID Service] 태그 확장 [개요](https://experienceleague.adobe.com/docs/experience-platform/tags/extensions/client/id-service/overview.html?lang=ko-KR)&#x200B;(방문자 ID 서비스를 구현하는 확장)을 참조하십시오.
