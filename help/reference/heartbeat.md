# 비디오 하트비트 및 Adobe 방문자 ID 서비스 {#video-heartbeat-and-the-experience-cloud-id-service}

비디오 하트비트를 사용하려면 방문자 ID 서비스가 필요합니다. JavaScript 또는 Flash에서 비디오를 측정할 때 사이트 페이지 및 비디오 재생에서 방문자 ID 서비스 를 구현하면 방문자가 비디오를 볼 때 별도로 카운트되지 않습니다.
