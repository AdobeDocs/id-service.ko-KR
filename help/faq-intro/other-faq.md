# 기타 CX 엔터프라이즈 솔루션에 대한 FAQ{#faqs-for-other-experience-cloud-solutions}

방문자 ID 서비스에서 다른 CX 엔터프라이즈 솔루션 사용과 관련된 기능 및 문제에 대한 FAQ입니다.

## Analytics 및 Audience Manager {#section-b3dd206d497041acb04554c6fb1c912a}

**방문자 ID 서비스를 구현하면 사용자의 방문 기록이 Adobe Analytics에서 Audience Manager으로 내보내집니까?**

다음 두 가지 옵션을 사용할 수 있습니다.

* 방문자 ID 서비스가 구현된 후에 사용자에게 방문한 활동이 있으면 방문자 및 방문자 기록이 Audience Manager으로 데이터 내보내기에 포함됩니다.
* 방문자 ID 서비스가 구현된 후에 사용자에게 방문한 활동이 없는 경우, 방문자 및 방문자 기록은 Audience Manager으로의 데이터 내보내기에 포함되지 않습니다. 새 활동이 없기 때문에 Analytics ID를 ECID에 연결할 방법이 없습니다.
